# 浦汇未来 AI 短视频生产系统

这是一套用本地 Agent Skills 批量制作产品短视频的工作区。主线是：拆解参考视频、确认产品事实、生成受控变体、人工验收，再输出成片。

- 当前主项目：[防尘膜短视频](03_视频项目/防尘膜短视频/README.md)
- 当前场景测试：[场景单元总表](03_视频项目/防尘膜短视频/08_场景单元/README.md)
- 可复用生产工具：[video-content-skills](90_视频生产工具/video-content-skills/README.md)

## 生产链路

```text
授权参考成片 + 产品事实 + 产品图
  → 参考片拆解
  → 母版脚本与 3～5 个受控变体
  → 关键帧生成与人工复核
  → 视频生成
  → 旁白、字幕和后期
  → 技术验证 + 画面验收
  → 20～60 秒成片
```

场景单元测试是这条链路中的“素材稳定性测试”：每次固定一个物体、承载面、环境、首帧和提示词，生成 3 条 10 秒素材检查随机废片率。

## 大文件与凭证

- MP4/MOV/MKV、WAV/MP3/M4A、PPTX 默认不上传 GitHub。
- GitHub 保留脚本、参数、关键帧、对比图和验收报告。
- API Key、Cookie、验证码使用环境变量或 macOS Keychain，不写入仓库。
- 付费生成必须先 dry-run，并设置图片任务数、视频任务数和付费秒数上限。
