# 覆盖完成态素材首轮验证

> 验证日期：2026-07-16（America/New_York）

## 目标

验证“减少施工动作、只展示产品已经覆盖物体的完成状态”是否能降低 Grok 视频素材的严重物理错误和整段废片率。

本轮只测一个场景单元，同一张关键帧、同一条视频提示词、同一模型和参数连续生成 3 次。三条之间唯一的计划外变量是模型随机性。

## 场景单元

| 维度 | 取值 |
| --- | --- |
| 物体种类 | 壁挂电视 |
| 物体形态 | 55 英寸、现代黑色矩形外观 |
| 环境场景 | 局部装修中的现代客厅 |
| 产品状态 | 一体式透明 PE 防尘膜已经完整覆盖电视，黄色胶条位于膜顶部 |
| 人物与动作 | 无人物、无安装、无拉膜、无撕除 |
| 景别 | 中景、视平线 |
| 镜头运动 | 极慢推进 |
| 输出 | 9:16、720×1280、10 秒、24 fps |

完整正向和负向提示词以 [`project.json`](project.json) 为权威来源。
三个变体在约 1 秒、5 秒、9 秒的对照见 [`review-contact-sheet.jpg`](review-contact-sheet.jpg)，每一行对应一个变体。

## 视觉验收

严格产品边界：黄色美纹胶必须只作为透明 PE 膜顶部的一条水平连接边存在，不能凭空增加第二条胶带或变成包边结构。

| 变体 | 严重物理崩坏 | 产品结构 | 时间连续性 | 严格结论 |
| --- | --- | --- | --- | --- |
| `variant-01` | 无 | 左侧凭空出现竖向黄色胶条 | 稳定 | 判废 |
| `variant-02` | 无 | 左侧凭空出现竖向黄色胶条 | 稳定 | 判废 |
| `variant-03` | 无 | 顶部水平胶条保持正确，无额外胶条 | 稳定 | 可用 |

共同优点：

- 没有手部、拉膜、撕膜和安装顺序错误；
- 电视没有明显变形、移动或穿模；
- 透明膜没有消失、增殖、融化或剧烈逆重力摆动；
- 三条都保持单镜头和缓慢推进，没有跳切。

共同风险：透明膜偏灰、褶皱较密，画面更像“施工保护状态”，不适合承担产品透明度特写，但可承担装修环境 B-roll。

## 首轮结论

- API 任务成功率：3/3；
- 技术可播放率：3/3；
- 严重物理崩坏率：0/3；
- 严格产品结构通过率：1/3；
- 严格可用素材率：1/3（33%）。

首轮支持“覆盖完成态比复杂施工动作更容易保持基本物理稳定”的方向，但还不能证明已经达到低废片率。当前主要错误已经从整段动作崩坏收敛为局部产品结构漂移；下一轮应在相同场景中强化：`only one horizontal yellow strip at the top; absolutely no yellow tape on either side or bottom edge`，再生成 3 条对照，观察严格通过率能否从 1/3 提高。

本轮不追加生成，避免在没有改提示词前重复消耗相同预算。
