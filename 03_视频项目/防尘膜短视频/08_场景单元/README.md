## 严格通过条件

1. 黄色胶带的承载面必须真实可见；
2. 透明膜必须从胶带边缘连续展开；
3. 不得凭空长出第二条胶带或错贴到物体上；
4. 物体、膜、胶带在 10 秒内不得变形、消失、融化或跳变；
5. 技术可播放只是底线，不代表画面合格。
